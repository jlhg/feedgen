# feedgen

Convert website content to Feed.

## Getting started

Start the web app by Docker Compose:

```
docker-compose up -d
```

Then go to `http://localhost:8080/<site>`.

## Supported sites

- `hnbest`: Top links on Hacker News
- `ptt`: 批踢踢實業坊
