# feedgen

Convert website content to RSS Feed.

## Getting Started

Start the web app by docker-compose:

```
docker-compose up -d
```

Then go to `http://localhost:8080/<site>`.

## Supported Sites

- `hackernews`: Top links on Hacker News
- `ptt`: 批踢踢實業坊
- `gamer_forum`: 巴哈姆特-哈拉區
